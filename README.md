# Project Car Rental

## Overview
Welcome to the **Car Rental System** repository! This project is a feature-rich car rental application designed to simplify and enhance the car rental process for both businesses and users. By leveraging modern technology, this platform enables seamless management of car rentals, availability tracking, and user bookings.

This project has been developed as part of my portfolio to demonstrate my technical skills and problem-solving capabilities. It highlights expertise in software development, design patterns, and integrating multiple technologies successfully in a single project.

---

## Key Features

### For Users
- **Registration & Login**: Secure authentication system for registering and logging in.
- **Search & Browse Cars**: Intuitive search functionality to explore available cars based on categories, pricing, and availability.
- **Booking System**: Users can book cars for a specified period based on real-time availability.
- **Responsive Design**: Fully functional on desktop, tablet, and mobile devices.

### For Admins
- **Dashboard**: A control panel to manage cars, rentals, and registered users.
- **Car Management**: Add, update, or remove cars from the fleet.
- **Booking Management**: Keep track of customer bookings and availability of vehicles.
- **Reports**: Generate reports on rental performance and usage statistics.

## Tech Stack
This project was built using the following technologies:

### Frontend
- **JavaScript**: Core scripting language used for interactivity and logic.
- **SCSS**: Stylesheet language used for designing and customizing the application's UI for a modern and responsive appearance.
- **HTML & CSS**: For structuring and designing lightweight and accessible web pages.

### Backend
- **C# (ASP.NET)**: Backend logic and API endpoints, enabling the project to serve as a robust and scalable backend solution.

---

## Workflow

The project workflow is designed to ensure clarity and ease of use:

1. **User Registration/Login**
   - New users need to register with their details.
   - Returning users can log in securely with their credentials.
2. **Car Browsing**
   - Users can browse the car catalog by applying filters such as brand, price, or availability.
3. **Booking**
   - Once a suitable car is found, users specify the rental period and make a booking.
4. **Admin Management**
   - Admins can log in to a dedicated dashboard to manage car inventories and oversee bookings.
5. **Payment Integration (Optional Future Addition)**
   - Could integrate online payment to confirm car bookings in real-time.

## Installation Instructions

Follow these steps to run the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Adarsh7509/Project-Car-Rental.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Project-Car-Rental
   ```

3. Install the necessary dependencies for both the frontend and backend.

4. Run the backend server:
   ```bash
   # Command to start the ASP.NET backend
   dotnet run
   ```

5. Open the project in your browser:
   ```bash
   http://localhost:5000
   ```
